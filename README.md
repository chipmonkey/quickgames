# Quick Arcade

Three simple browser games — all in a single HTML file, no dependencies, no build step.

Open `index.html` in any browser or run `serve .` locally.

- **Asteroids** — destroy asteroids, avoid collisions, survive
- **Tetris** — classic block-dropping puzzle with ghost piece and hard drop
- **Robotron** — multidirectional shooter with wobble, zoomer, and shooter enemies

These games are meant to demonstrate core JavaScript game development: canvas rendering, game loops, collision detection, input handling, and state management. Fork, tweak, and see what happens.
